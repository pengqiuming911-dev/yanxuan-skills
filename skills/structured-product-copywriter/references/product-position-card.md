# 产品点位卡生成流程

“产品派息与敲出观察点位表图片”必须来自通毓终端产品点位页的“复制为图片”功能。

页面：
`https://terminal.tongyu-quant.com/smallTool/index.html#/product-position`

## 硬规则

1. 打开产品点位页。
2. 选择产品类型并填入产品基础要素。
3. 点击提交，等待结果卡生成。
4. 点击页面自带的“复制为图片”按钮。
5. 从浏览器剪贴板读取 `image/png`，保存为产品点位图。
6. 保存后只做近白色 trim/crop，删除边缘空白；不得改变原图表格样式、背景色、红色重点字体。

不能用本地 HTML 重画表格，不能用浏览器视口截图代替，不能静默兜底成自制表格。

若剪贴板里没有 `image/png`，应显式报错或在 manifest 中写 `[图片待补: 产品派息与敲出观察点位表图片]`。

## 填表口径

### 产品信息录入

- **产品名称**：填 `产品名称简写-结构名称`，与飞书文档标题一致，如 `泰创纶哲CTA一期-2倍DCN`。
- **产品类型**：本页（产品点位卡）只有「锁盈」「DCN」两个选项，按给定的结构类型自动选——结构名含 `DCN` 选 DCN，其余（经典/早利/多倍锁盈/全保锁盈/雪球类）选锁盈。注意：胜率回测页的"基础结构"有 5 类（经典/凤凰/早利/蝶变/DCN），那是另一套，别混到产品卡这里——产品卡只二分。早利 vs 经典在本页只影响票息分段填法（早利按区间填、经典单一/一致票息），不改变类型下拉。不要选错类型，否则后续票息/敲出表单字段不同。

### 基础信息

- **期限 / 锁定期解析**：把用户给的 `期限` 字符串拆成两个数。
  - `36M锁3M` → 期限 = 36，锁定期 = 3。
  - `36M无锁` → 期限 = 36，锁定期 = 1（无锁 = 锁定期填 1，不是 0）。
  - 规则：`<N>M锁<L>M` 取 N 与 L；`<N>M无锁` 取 N，锁定期填 1。
- **降落伞月份**：一律填 1。
- **期初敲出线 / 降敲 / 降落伞 / 派息线 / 保证金**：按结构要素原值填。
- **末次观察敲出价**：填**和降落伞(=期末障碍价)一致的值**，覆盖表单默认自动联动的末次敲出线。本产品末次观察敲出价 = 期末障碍价 = 降落伞（如 60），三者一致——`tongyu_left_panel.py` 已在期末障碍价后紧接着填 `末次观察敲出价 = args.parachute`。若该结构无此字段（`fill_field` 返回 NOTFOUND/NOINPUT）忽略即可。
- **挂钩标的**：按结构标的选（中证1000/沪深300/中证500/创业板指等）。

### 票息设置（按产品类型分支）

- **DCN**：用「每月或有派息」填月票息，如 `1.39`（费后派息原值，不乘 12）。
- **锁盈**：按区间年化票息录入。例：`前18个月票息36%,后18个月票息1%` → 区间1：`1-18`：`36`；区间2：`19-36`：`1`。区间月份上下界按期限拆分写入。

### 入场信息

- **入场日期**：按生成物料当前的日期（当天）。`product_card.py` / `product_card_locky.py` 不传 `--entry-date` 时默认 `date.today()`，产品卡即当前日期口径。产品卡按「as of today」快照生成，入场日期固定取当天，不要用产品未来实际入场日覆盖。
- **入场点位**：按生成物料当前的点位——用 `fetch_quote.py` 取到的真实现价（当天），不要用记忆里的旧点位；用户明确给了当天点位就用用户的。

## 装进飞书云文档

最终装配用：

```bash
python3 scripts/create_feishu_doc.py --manifest manifest.json --title "产品简称-结构名称"
```

manifest 第 5 节写：

```json
{"type":"subheading","text":"产品派息与敲出观察点位表图片"},
{"type":"image","path":"assets/product-card.png","caption":"产品派息与敲出观察点位表图片"}
```

不要默认使用 `scripts/build_docx.py`，不要返回 `/file/` 链接。

## 验收

最终图片应与通毓复制出的原始 PNG 风格一致，保留清晰背景色和红色重点字体，底部没有表格外大块空白。
